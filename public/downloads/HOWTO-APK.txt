Place your signed release APK here as:

  laundroswipe-print-bridge.apk

The Printers page button uses /api/print-bridge-apk which redirects to:

  /downloads/laundroswipe-print-bridge.apk

so the file is served as a normal static asset (fine for large APKs on Vercel).

Alternatively set NEXT_PUBLIC_PRINT_BRIDGE_APK_URL (browser opens that URL directly)
or PRINT_BRIDGE_APK_URL (server-only redirect from /api/print-bridge-apk).
