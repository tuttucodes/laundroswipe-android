Add these icon files in this folder (public/) for "Add to Home Screen":

  icon-192.png        192 x 192 px   (Android & manifest)
  icon-512.png        512 x 512 px   (Android install / splash)
  apple-touch-icon.png  180 x 180 px (iOS home screen)

Use the same logo/icon for all. PNG format.
If you only add one, add apple-touch-icon.png for iOS.

See deployment.md section "PWA / Add to Home Screen" for full steps.
